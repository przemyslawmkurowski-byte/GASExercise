# GASExercise
Implementacja okrojonego combat engine'u gry tabletop RPG Fabula Ultima z użyciem Gameplay Ability System.

## Cel ćwiczenia
Zapoznanie z GASem. Zbadanie sensowności użycia GASa w grach typu jRPG.

## Dlaczego Fabula Ultima, a nie dowolny jRPG?
1. Starter do Fabuli mam pod ręką.
2. Ze względu na swoją naturę (ttrpg udający jrpg, będący realnie gdzieś pomiędzy), Fabula oferuje interesujące wyzwanie implementacyjne.

## Dlaczego GAS?
1. Ze względu na popularność.

## Zakres ćwiczenia
Ćwiczenie obejmuje pojedynczą walkę między drużyną bohaterów (kontrolowanych przez gracza), a drużyną NPC (kontrolowanych przez prymitywne AI). W ćwiczeniu można wyróżnić:
1. Integrację GAS - atrybuty, efekty, abilities i relacje między nimi
2. Subsystem walki turowej - określający, który z Pawnów ma prawo wykonać akcję
3. Subsystem AI.
